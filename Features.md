# VRSTE Features

### 1. Write and Run Real C++ Code in the Browser
- Monaco-powered editor
- Real C++ (not a simplified subset)
- Live transpilation to JavaScript
- Async runtime execution
- Robot control in real-time on the field

### 2. Engine-Agnostic Robot API
- Motor power control
- Encoder simulation
- Heading and odometry
- Timing functions
- Autonomous helpers
- Sensor simulation (planned)
- Works consistently across all physics engines

### 3. Modular Physics Engine System
- Physics Abstraction Layer supports multiple engines:
  - Simulation.js (lightweight, built-in)
  - MuJoCo (high-fidelity physics)
  - Bullet (rigid-body dynamics)
  - Drake (research-grade modeling)
- Switch engines without breaking user code
- Suitable for students, competitive teams, researchers, and professionals

### 4. Coordinate Plane Mode
- Full X/Y axes and gridlines
- Real-time pose visualization
- Unit switching (inches or meters)
- Supports path planning, motion profiling, pure pursuit, and Ramsete-style controllers

### 5. Real-Time Field Rendering
- Smooth robot motion
- Accurate heading visualization
- Custom field layouts
- Future 3D rendering support

### 6. Browser-Native, Zero Installation
- No downloads or OS dependencies
- Works on Windows, macOS, Linux, and Chromebooks
- Low barrier to entry for teams and classrooms

### 7. Extensible by Design
- Add physics engines, robot models, sensors, field types
- Support for plugins and extensions
- Contributors can enhance VRSTE without modifying the core

### 8. Built for Both Education and Professional Use
- Serves students, competitive teams, university courses, researchers, and professional engineers
- Bridges educational simulators and complex professional tools

### 9. A Platform, Not Just a Simulator
- Community contributions and open-source development
- Shared autonomous routines
- Expandable physics backends
- Growing library of robotics tools
